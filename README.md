# TaskFlow-Pro
Full-Stack Web Application - TaskFlow Pro

TaskFlow Pro is a full-stack project management application designed for modern development teams. It provides a collaborative platform for task management, project tracking, and team coordination with real-time updates.

Features
User Authentication & Authorization - JWT-based auth with role-based access control

Project Management - Create, organize, and track multiple projects

Task Board - Kanban-style board with drag-and-drop functionality

Real-time Updates - Live task updates and notifications via WebSocket

File Attachments - Upload documents and images to tasks

Activity Log - Complete audit trail of all actions

Team Collaboration - Invite team members with different permission levels

Advanced Search - Full-text search across tasks and projects

Email Notifications - Daily digests and assignment alerts

RESTful API - Fully documented API for integration

Tech Stack
Frontend

Redux Toolkit - State management

Tailwind CSS - Utility-first CSS framework
